自定义按钮:
KSButtonOptions
title   style   onClick


1. 创建option
option: KSDialogOptions = {
  message: '告知当前状态，信息和解决方法如果文字换行的情况',
  okButton: {
    title: '确定',
    style:KSDialogButtonStyle.Guide,
    onClick: () => {
      if (option.dialogID) {
        KSCustomDialogOption.close(option.dialogID)
      }
    }
  },
}
2. 调用API创建弹窗
DialogBuilderFactory.showSimpleDialog(this.option, this)
